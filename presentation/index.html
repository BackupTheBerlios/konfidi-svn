Identity with PGP

  • Public/private keypairs
  • "Web of trust"
  • keyservers
  • multipart/signed emails

Topical Trust

  • Person A trusts Person B on topic T at value X
  • RDF
    • XML markup
    • Subject-predicate-object statements
    • FOAF subjects
    • Trust schemas
    • no servers

FOAF Server

  • Public server, like PGP keyserver
  • Stores FOAF and related RDF data (e.g. trust)
  • You must PGP sign your file

Trust Server

  • Data synchronized from FOAF server
  • Recursive computations
  • Models
    • Relative to source
    • Scale
    • Distrust

PGP Pathfinder

  • Verify identity before trust computation
  • Use wotsap data files

Frontend

  • Public API for client queries
  • Invokes pathfinder
  • Invokes trust server

Clients

  • PGP and Gnu Privacy Guard
  • "Trust-o-matic" for FOAF server
  • MUAs
    • Sign outgoing emails
    • Validate incoming signatures
    • Query frontend with PGP fingerprints of sender & recipient
    • Add headers
    • Filter appropriately
